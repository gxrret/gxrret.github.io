GIF GURU

GIF Guru is an Alexa skill that leverages the Giphy API to search for and deliver GIF images to user’s phones. It uses Amazon S3 to secure sensitive information using at-rest encryption, Amazon DynamoDB to associate phone numbers with Echo devices for data persistence, and Amazon SNS to text resulting GIF’s to users. All API calls were asynchronous and were chained together using promise objects.

Skill Page

MY BOXING COACH

My Boxing Coach is a skill that allows Alexa to voice out boxing commands to users. This skill is great for boxers of all levels to practice when they don’t have the luxury of a class, coach, or partner. The user can tell Alexa pre or mid-interaction how many rounds they would like to do. Each round consists of combinations that increase in difficulty, but no two rounds are the same. My Boxing Coach boasts the highest usage rate of all my skills with over 120 unique users per month.
